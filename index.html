襄派海外仓 (XiangPai) 价格预估模型

分段预估（头程 / 入库上架 / 仓储 / 出库 / 尾程 / 退货换标）+ 全链路汇总 ｜ 覆盖：美国本地仓 · 退货换标+尾程专长 ｜ 评级 🟢已验证 / 🟡报价完整·待账单核 / 🔴未对齐 / ⚪无数据

数据覆盖：入库费按重🟡 · 双轨仓储(体积+重量)🟡 · 出库操作费按重🟡 · 尾程双渠道UPS+USPS🟡 · 退货换标(退货处理+换标$0.5+重包装+箱标)🟡

场景输入（改任一项点「计算预估」实时重算；切上方标签看各段 / 全链路）

尾程计费重=max(实重, 材积/6000)；UPS 表按 lbs、USPS 表按 kg，均向上进位取档。仓储双轨：体积费$/天 + 重量费$/kg/天，按库龄季度(首月/第2-3月/第2季/第3季/第4季/第5季)取费率。退货换标段含退货处理费+换标$0.5/件+重包装+箱标。头程非本仓报价范围，手填整批USD即纳入全链路。
📌 襄派报价完整覆盖入库/仓储/出库/尾程(双渠道)/退货换标，模型各科目标 🟡「报价完整·待账单核」——因暂未接入襄派真实账单逐项验证。尾程默认 UPS Ground（经济地面派送），可切 USPS 美西同档对比；美西=Zone8。头程需另按渠道报价手填。
